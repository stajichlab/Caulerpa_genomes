Caulerpa Genomes
====

Genome assembly and annotation of Caulerpa lentillifera and C. racemosa as part of Zheng et al. 2026. Curr Biol. doi: [10.1016/j.cub.2026.03.033](https://doi.org/10.1016/j.cub.2026.03.033)

This includes the pipelines for genome annotation ([Funannotate](https://funannotate.readthedocs.io/en/latest/) v1).
